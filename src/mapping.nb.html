How to populate eco:isTaxonomicScopeComplete

The field eco:isTaxonomicScopeComplete is difficult to populate manually, so I am populating it with the script below.

If all dwc:Organism of an dwc:Occurrence from an dwc:Event are within eco:targetTaxonomicScope, set eco:isTaxonomicScopeComplete to true.

# Perform left join on humboldt and occurrence tables
taxonomic_scope_joined_table <- humboldt %>%
  left_join(occ, by = "eventID", relationship = "many-to-many") %>%
  left_join(target, by = "scientificName", relationship = "many-to-many") %>%
  group_by(eventID, isTaxonomicScopeComplete) %>%
  # if all dwc:Organism of dwc:Occurrence of an dwc:Event are in targetTaxonomicScope
  mutate(isTaxonomicScopeComplete = all(!is.na(scientificName))) %>%  
  select(eventID, isTaxonomicScopeComplete) %>%
  unique()

# Update the original humboldt table
humboldt$isTaxonomicScopeComplete <- taxonomic_scope_joined_table$isTaxonomicScopeComplete
humboldt %>% select(eventID, isTaxonomicScopeComplete) %>% head()

dwc:occurrenceStatus of presence-only dwc:Occurrence data

The dwc:scientificName in red are dwc:Organism caught during the dwc:Event, but are not within the eco:targetTaxonomicScope (by-catch); the ones in blue are within the eco:targetTaxonomicScope. In another words, in this dataset, the target is the thing the researchers intended to catch using the sampling design. As by-catch is not in eco:targetTaxonomicScope, we cannot make inference that they are absent from the dwc:Events

How to populate eco:isLifeStageScopeComplete

I have difficulty writing a script for eco:isLifeStageScopeComplete, so I did it manually.

conditions eco:isLifeStageScopeComplete
if dwc:lifeStage of an dwc:Organism from eco:targetTaxonomicScope is not recorded, but eco:targetLifeStageScope == all true

What not to do

If you know what are the life stages applicable for your target taxa, please list them all out instead of using all. This allows user to make inference about the non-detection of the taxon at its specific life stage.

Remarks

  • difference between eco:taxonCompletenessReported and eco:isTaxonomicScopeComplete ?
  • completeness (eco:isLifeStageScopeComplete) per event instead of per target taxa is limiting the information that can be inferred.
  • how to fill in eco:excludedLifeStageScope when there is none (eco:targetLifeStageScope == all life stages). Because of the way the data is ordered alongside with eco:targetTaxonomicScope. Cant make inference if there is only eco:targetLifeStageScope without explicit eco:excludedLifeStageScope
  • lack of identifiers for dwc:scientificName in eco:targetTaxonomicScope may be problematic for homonyms.
  • the mapping was done to explore dwciri of terms could be used in IPT: https://github.com/gbif/ipt/issues/1947
